Arduino Practice [![Build Status]][travis]
========
Please use [PlatformIO](http://platformio.org/) to build and execute project.
```bash
# Build project
platformio run

# Upload to connected Arduono machine
platformio run -t upload

# Monitor serialports
platformio serialports monitor
```
